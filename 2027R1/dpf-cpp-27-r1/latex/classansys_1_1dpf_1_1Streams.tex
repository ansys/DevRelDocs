\doxysubsection{Detailed Description}
Defines an open, ready-\/to-\/use, data sources. 

\doxylink{classansys_1_1dpf_1_1Streams}{Streams} is an entity containing the data sources. Once the files in the streams are opened, they stay opened and they keep some data in cache to make the next evaluations faster. To close the files, release the streams.
