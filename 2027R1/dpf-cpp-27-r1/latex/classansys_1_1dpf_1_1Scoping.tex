\doxysubsection{Detailed Description}
Define a set of entities by \doxylink{classansys_1_1dpf_1_1Scoping_acf26749e29d9f23de6c70806ccfaea6b}{ids}. 

The scoping is entities ids representing a subset of the model\textquotesingle{}s support. Typically, scoping can represent node ids, element ids, time steps, frequencies, joints... Its location indicates what kind of entity the scoping is referring to. Scopings are used to identify the entities where a field is scoped or to choose (through an input pin) a subset on which an operator should compute its result. \begin{Desc}
