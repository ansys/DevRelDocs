\doxysubsection{Detailed Description}
Holds string data describing a Property located on given entities (the \doxylink{classansys_1_1dpf_1_1StringField}{String\+Field}\textquotesingle{}s data is bounded with its scoping). Is the equivalent of the \doxylink{classansys_1_1dpf_1_1Field}{Field} but with string values data (and not double values).
