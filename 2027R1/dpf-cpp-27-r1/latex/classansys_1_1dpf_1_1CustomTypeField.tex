\doxysubsection{Detailed Description}
Holds data located on given entities (the \doxylink{classansys_1_1dpf_1_1Field}{Field}\textquotesingle{}s data is bounded with its scoping). The unitary data type can be chosen by the user (it can be int32, float, char...). Is the equivalent of the \doxylink{classansys_1_1dpf_1_1Field}{Field} but with custom type values data. (and not double values). \begin{Desc}
