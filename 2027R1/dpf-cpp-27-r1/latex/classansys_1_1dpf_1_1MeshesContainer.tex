\doxysubsection{Detailed Description}
Contains a group of meshes. The meshes container is designed as a set of meshes ordered through labels and ids. Labels identify how the fields are filtered. The most common meshes container have the label "{}body"{} with ids corresponding to each body entity. \begin{DoxySeeAlso}{See also}
