\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class ansys::\+dpf::\+\+Dpf\+Vector$<$ T $>$}
Represents a modifiable vector of data and manages its memory. 

If the \doxylink{classansys_1_1dpf_1_1DpfVector}{Dpf\+Vector} is returned by a local entity, then it\textquotesingle{}s a direct pointer to its data else if the \doxylink{classansys_1_1dpf_1_1DpfVector}{Dpf\+Vector} is returned by a remote entity, then it\textquotesingle{}s a copy of the data and the remote data is updated (if necessary) when the \doxylink{classansys_1_1dpf_1_1DpfVector}{Dpf\+Vector} is destructed.
