\doxysubsection{Detailed Description}
Holds the mesh for a given region (body, faces, skin, ...). 

The meshed region is an entity describing a mesh. Node and element scopings, element types, connectivity (list of node indices composing each element) and node coordinates are the fundamental entities composing the meshed region. It can also have materials, named selections... \begin{Desc}
