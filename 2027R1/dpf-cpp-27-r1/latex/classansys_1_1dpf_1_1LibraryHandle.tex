\doxysubsection{Detailed Description}
Provides an access to use DPF HGP API. The instance of \doxylink{classansys_1_1dpf_1_1LibraryHandle}{Library\+Handle} is only needed once. 

Wraps data\+Processing\+Core loading in an RAII (Resource acquisition is initialization) structure in order to ensure correct loading/\+unloading of the DPF core functionalities. Must be the first call made to DPF API.
