\doxysubsection{Detailed Description}
\doxylink{classansys_1_1dpf_1_1Client}{Client} for remote DPF connection. A \doxylink{classansys_1_1dpf_1_1Client}{Client} instance is always associated to a DPF server process. The corresponding server needs to run so that the client instance can then be correctly used. This client allows to create DPF objects on a remote server.
