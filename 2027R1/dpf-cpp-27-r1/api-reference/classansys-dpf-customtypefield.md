# Class ansys::dpf::CustomTypeField

<a id="classansys-dpf-customtypefield"></a>

![][C++]
![][public]



Holds data located on given entities (the [Field](classansys-dpf-field.md#classansys-dpf-field)'s data is bounded with its scoping). The unitary data type can be chosen by the user (it can be int32, float, char...). Is the equivalent of the [Field](classansys-dpf-field.md#classansys-dpf-field) but with custom type values data. (and not double values).

**Inherits from**:

* [ansys::dpf::DpfTypes](classansys-dpf-dpftypes.md#classansys-dpf-dpftypes)
