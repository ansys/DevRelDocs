# Class ansys::dpf::StringField

<a id="classansys-dpf-stringfield"></a>

![][C++]
![][public]



Holds string data describing a Property located on given entities (the [StringField](classansys-dpf-stringfield.md#classansys-dpf-stringfield)'s data is bounded with its scoping). Is the equivalent of the [Field](classansys-dpf-field.md#classansys-dpf-field) but with string values data (and not double values).

**Inherits from**:

* [ansys::dpf::DpfTypes](classansys-dpf-dpftypes.md#classansys-dpf-dpftypes)
