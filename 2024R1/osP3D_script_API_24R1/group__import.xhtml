◆ importMesh_STL()

MetaStructure importMesh_STL ( string  filename,
bool  merge_nodes = true,
number  relative_node_distance = 1e-4 
)

imports the mesh from a single STL file (either ASCII or binary)

Parameters
filenamedefines the input file
Returns
a meta structure containing the mesh
Parameters
merge_nodesDefines if nodes with nearly identrical coordinates are merged. If false, each node belongs to eactly one element. If true, one may erroneously merge nodes which do not belong together (eg. if two separate bodies are very close to each other)
relative_node_distanceThe minimum relative distance between two nodes. If a distance between two nodes is below the tolerance, the nodes will be merged. The distance is defined by this factor multiplied with the largest triangle edge length in the model.
