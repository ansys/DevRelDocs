the_scalar_datathe subset of scalar data to be token as input. All quantity idents are extracted. Then all n field quantities is put into correlation with all m scalar quantities (n*m new objects will be computed!). As a default, the new correlation field objects are appended to the field data quantities as individual designs. The design name is "correlation[scalar_quantity_ident]".
