# SurfacePolygonSigma 

## Description 

Return value: returns an array of 2 VARIANT representing of the sigma x and y values for the given polygonal area.

## Syntax 

*object*.SurfacePolygonSigma\(VARIANT\* vPtX, VARIANT\* vPtY\) As Variant

- *object*: Virtual Photometric Lab object

- *vPtX*: array of VARIANT for the horizontal coordinate of each vertex of the polygon

- *vPtY*: array of VARIANT for the vertical coordinate of each vertex of the polygon
