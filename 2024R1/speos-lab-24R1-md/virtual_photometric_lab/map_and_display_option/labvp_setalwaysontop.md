# SetAlwaysOnTop 

## Description 

Sets the window of the Virtual Photometric Lab on top \(or use the normal behavior\). Returns a value different from 0 if no error has occurred.

## Syntax 

*object*.SetAlwaysOnTop\(short bAlwaysOnTop\) As Short

- *object*: Virtual Photometric Lab object

- *bAlwaysOnTop*: A value different than 0 to set the window always on top
