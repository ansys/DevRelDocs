# show

<a id="ansys.meshing.prime.graphics.PrimePlotter.show"></a>

#### PrimePlotter.show(plottable_object=None, screenshot=None, name_filter=None, scope=None, \*\*plotting_options)

Show the plotted objects.

**Parameters:**

  **plottable_object**

  Object to show.

  **screenshot**

  Path to save a screenshot to.

  **name_filter**

  Regular expression with the desired name or names to include in the plotter.

  **\*\*plotting_options**

  Keyword arguments. For allowable keyword arguments, see the
    `Plotter.add_mesh` method.
    Options only applied to PyVista plottable objects.

**Return type:**

  [`None`](https://docs.python.org/3.11/library/constants.html#None)
