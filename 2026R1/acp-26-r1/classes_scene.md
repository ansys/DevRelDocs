# Scene classes

These classes represent scene and view data used for visualization within a compolyx model.

| Name | Description |
|--------------------------------------------------------------------------------------------------------|--------------------------------------------|
| [`Scene`](_autosummary/compolyx.Scene.md#compolyx.Scene)(graph[, model, name, id, title, view, ...])   | Represents a scene.                  |
| [`View`](_autosummary/compolyx.View.md#compolyx.View)(name[, position, orientation, ...])              | Represents a view and its display properties. |
