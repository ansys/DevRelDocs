#### ModelingPly.add_taper_edge(taper_edge, angle, offset=0.0)

Add taper edge to modeling ply.

* **Parameters:**
  - taper_edge: The taper edge to be added to the modeling ply.
  - angle: Angle for tapering.
  - offset: Offset for tapering.
