# Model classes

The following classes represent model, geometry, ply, selection, and analysis data within a ComPoLyX model. Each class corresponds to a specific object type and provides access to that data.

| Name | Description |
|--------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------|
| [`AnalysisPly`](_autosummary/compolyx.AnalysisPly.md#compolyx.AnalysisPly)(graph, obj[, parent])                                                 | Represents an analysis ply.                                                            |
| [`BooleanSelectionRule`](_autosummary/compolyx.BooleanSelectionRule.md#compolyx.BooleanSelectionRule)(graph, obj[, parent])                      | Represents a boolean selection rule.                                                                              |
| [`ButtJointSequence`](_autosummary/compolyx.ButtJointSequence.md#compolyx.ButtJointSequence)(graph, obj[, parent])                               | Represents a butt-joint sequence.                                                          |
| [`CADCompound`](_autosummary/compolyx.CADCompound.md#compolyx.CADCompound)(graph, obj[, parent])                                                 | Represents a CAD compound entity.                                                            |
| [`CADFace`](_autosummary/compolyx.CADFace.md#compolyx.CADFace)(graph, obj[, parent])                                                             | Represents a CAD face.                                                          |
| [`CADGeometry`](_autosummary/compolyx.CADGeometry.md#compolyx.CADGeometry)(graph, obj[, ...])                                                    | Represents CAD geometry data.                                                                                                     |
| [`CADReference`](_autosummary/compolyx.CADReference.md#compolyx.CADReference)(graph, obj[, parent])                                              | Represents a CAD reference.                       |
| [`CADShell`](_autosummary/compolyx.CADShell.md#compolyx.CADShell)(graph, obj[, parent])                                                          | Represents a CAD shell.                                                                |

| [`Chapter`](_autosummary/compolyx.Chapter.md#compolyx.Chapter)(name, parent[, view, ply_entities, ...])                                          | Represents a plybook chapter.                                                                 |

| [`CylindricalSelectionRule`](_autosummary/compolyx.CylindricalSelectionRule.md#compolyx.CylindricalSelectionRule)(graph, obj[, parent])          | Represents a cylindrical selection rule.                                                                         |
| [`EdgeSet`](_autosummary/compolyx.EdgeSet.md#compolyx.EdgeSet)(graph[, obj, parent])                                                             | Represents an edge set.                                                                                    |

| [`ImportedModelingGroup`](_autosummary/compolyx.ImportedModelingGroup.md#compolyx.ImportedModelingGroup)(graph, obj[, parent])                   | Represents an imported modeling group.                                                                                                     |
