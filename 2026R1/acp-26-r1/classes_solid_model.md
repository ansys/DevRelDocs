# Solid-model classes

These classes represent solid-model geometry, guides, and element sets used within a ComPoLyX model.

| Name | Description |
|-----------------------------------------------------------------------------------------------------------------------|------------------------------------------------------|
| [`CutOffGeometry`](_autosummary/compolyx.CutOffGeometry.md#compolyx.CutOffGeometry)(obj[, parent])                    | Represents cutoff geometry used for solid‑model operations.                           |
| [`ExtrusionGuide`](_autosummary/compolyx.ExtrusionGuide.md#compolyx.ExtrusionGuide)(obj[, parent])                    | Represents an extrusion guide.                              |
| [`ImportedSolidModel`](_autosummary/compolyx.ImportedSolidModel.md#compolyx.ImportedSolidModel)(graph, obj[, parent]) | Represents an imported solid model.                       |
| [`LayupMappingObject`](_autosummary/compolyx.LayupMappingObject.md#compolyx.LayupMappingObject)(graph, obj[, parent]) |  Represents a layup‑mapping object used in solid‑model workflows.                     |
| [`SnapToGeometry`](_autosummary/compolyx.SnapToGeometry.md#compolyx.SnapToGeometry)(obj, parent)                      | Represents a snap‑to‑geometry guide.                        |
| [`SolidElementSet`](_autosummary/compolyx.SolidElementSet.md#compolyx.SolidElementSet)(graph[, obj, parent])          | Represents an element set associated with a solid model. |
| [`SolidModel`](_autosummary/compolyx.SolidModel.md#compolyx.SolidModel)(graph, obj[, parent])                         | Represents a solid model.                            |
