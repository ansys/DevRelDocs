# Material classes

The following classes represent composite material data within a compolyx model. Each class represents one specific type of material-related object, such as a material, fabric, stackup, sublaminate, or property set.

| Name | Description |
|-----------------------------------------------------------------------------------------------------------|--------------------------------------------------|
| [`Fabric`](_autosummary/compolyx.Fabric.md#compolyx.Fabric)(graph, obj[, parent])                         | Represents a fabric material.                       |
| [`Material`](_autosummary/compolyx.Material.md#compolyx.Material)(graph, obj[, parent])                   | Represents a material defined in the model.                         |
| [`MaterialData`](_autosummary/compolyx.MaterialData.md#compolyx.MaterialData)(graph[, parent])            | Manages all composite material data in the model. |
| [`PropertySet`](_autosummary/compolyx.PropertySet.md#compolyx.PropertySet)(gil_wrapper[, parent_, name_]) | Represents a set of material properties.                         |
| [`Stackup`](_autosummary/compolyx.Stackup.md#compolyx.Stackup)(graph, obj[, parent])                      | Represents a stackup definition.                      |
| [`SubLaminate`](_autosummary/compolyx.SubLaminate.md#compolyx.SubLaminate)(graph, obj[, parent])          | Represents a sublaminate within a stackup.                  |
