# Plot classes

The following classes provide access to plot and visualization data in a compolyx model. Each class corresponds to a specific type of plot data, such as contour values, angles, thickness, and field results.

| Name | Description |
|---------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------|
| [`AngleData`](_autosummary/compolyx.plot_data.AngleData.md#compolyx.plot_data.AngleData)(graph, obj[, parent, display_data])                      | Represents plot data for angle-based results.                          |
| [`ContourData`](_autosummary/compolyx.plot_data.ContourData.md#compolyx.plot_data.ContourData)(graph, obj[, parent, display_data])                |  Represents contour plot data.                           |
| [`DrapingData`](_autosummary/compolyx.plot_data.DrapingData.md#compolyx.plot_data.DrapingData)(graph, obj[, parent])                              | Represents draping-related plot data.                          |
| [`FieldDefinitionData`](_autosummary/compolyx.plot_data.FieldDefinitionData.md#compolyx.plot_data.FieldDefinitionData)(graph, obj[, parent, ...]) | Represents plot data linked to a field definition.                          |
| [`LayupMappingData`](_autosummary/compolyx.plot_data.LayupMappingData.md#compolyx.plot_data.LayupMappingData)(graph, obj[, parent, ...])          | Represents plot data for layup mapping results.                          |
| [`LayupPlotDict`](_autosummary/compolyx.plot_data.LayupPlotDict.md#compolyx.plot_data.LayupPlotDict)(name[, label, obj, list, ...])               |  Container for layup plot objects. |
| [`PlotData`](_autosummary/compolyx.plot_data.PlotData.md#compolyx.plot_data.PlotData)(graph, obj[, parent])                                       | Base class for plot data objects.                          |
