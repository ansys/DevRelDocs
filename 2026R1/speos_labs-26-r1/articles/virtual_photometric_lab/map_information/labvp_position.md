# Position 

## Description 

Sets the position in a Virtual Photometric Lab/Virtual Human Vision Lab window. Returns 0 if no error has occurred, 1 otherwise 1.

## Syntax 

*object*.Position\(short nXPos, short nYPos\) As Short

- *object*: Virtual Photometric Lab or Virtual Human Vision Lab object

- *nXPos, nYPos*: new position of the window.
