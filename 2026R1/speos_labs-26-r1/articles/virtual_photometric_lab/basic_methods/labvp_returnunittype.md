# ReturnUnitType 

## Description 

Returns the unit type of the map.

## Syntax 

*object*.ReturnUnitType\(\) As Short

- *object*: Virtual Photometric Lab object

- Return value:

    0: radiometric

    1: photometric

    2: gain

    3: unknown

    4: FTM

    5: degree

    6: inverse meter

    7: inverse square meter

    8: percent

    9: diopter
