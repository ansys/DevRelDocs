# Operation 

## Description 

Defines the name of the operation. Returns 0 if the call succeeded, 1 otherwise.

## Syntax 

*object*.Operation\(BSTR bstrOperationName\) As Short

- *object*: Photometric Calc object

- *bstrOperationName*: operation name

    "Addition"

    "Subtraction"

    "ValueMultiplication"

    "MapMultiplication"

    "MapDivision"

    "SymetryVertical"

    "SymetryHorizontal"

    "VerticalFlip"

    "HorizontalFlip"

    "Rotation"

    "MergeResult"

    "MapUnion"

    "SpectrumMultiplication"

    "MapSignalToNoiseRatio"
