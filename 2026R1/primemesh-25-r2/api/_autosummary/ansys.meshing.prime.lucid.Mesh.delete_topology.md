# delete_topology

<a id="ansys.meshing.prime.lucid.Mesh.delete_topology"></a>

#### Mesh.delete_topology(part_expression='\*', delete_edges=True)

Delete topology in one or more parts.

* **Parameters:**

  **part_expression**
  : Expression of the parts where the topology must be deleted.
    The default is `"*"`.

  **delete_edges**
  : Whether to delete edge zonelets. The default is `True`.
