* **Z**: The Z coordinates for the specified triangle Vertex.
* **XNormal**: The Normal for the X coordinates for the specified triangle Vertex.
* **YNormal**: The Normal for the Y coordinates for the specified triangle Vertex.
* **ZNormal**: The Normal for the Z coordinates for the specified triangle Vertex.

* **Object**:
