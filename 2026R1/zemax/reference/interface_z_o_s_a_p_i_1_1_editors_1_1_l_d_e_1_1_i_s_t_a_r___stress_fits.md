### Function GetRetardanceMap

<a id="interface_z_o_s_a_p_i_1_1_editors_1_1_l_d_e_1_1_i_s_t_a_r___stress_fits_1a724d81d1351b5cdfc53882c5f99f19c4"></a>

![][public]


```csharp
ISTAR_PointWithRetardance[] ZOSAPI.Editors.LDE.ISTAR_StressFits.GetRetardanceMap(int density, int coordinateOption, int field, double Jx, double Jy, double phix, double phiy)
```


Traces a set of rays with hexapolar pattern to the back of the lens and calculate the phase retardance in radians experienced by each ray.

The OPD retardance in nm can be calculated by dividing the phase retardance by 2pi and multiplying by the wavelength in nm. 

**Parameters**:

* **density**: Density parameter used to generate the hexapolar pattern.
* **coordinateOption**: 1 to return positions in global coordinates. Otherwise positions are returned in local coordinates.
* **field**: Which field is used to generated the rays to be traced.
* **Jx**: X component of the Jones electric field.
* **Jy**: Y component of the Jones electric field.
* **phix**: Phase of the X component of the Jones electric field in degrees.
* **phiy**: Phase of the Y component of the Jones electric field in degrees.


**Returns**:

Each entry represents a ray reaching the back of the lens and contains its position and phase retardance in radians between the X and Y components of the associated electric field.


**Parameters**:

* int **density**
* int **coordinateOption**
* int **field**
* double **Jx**
* double **Jy**
* double **phix**
* double **phiy**

```csharp
ISTAR_PointWithIndexAndRetardance[] ZOSAPI.Editors.LDE.ISTAR_StressFits.GetPointRetardanceList(int density, int coordinateOption, int field)
```


Traces a set of rays with hexapolar pattern to the back of the lens. Calculates the index and point retardance (in radians / mm) experienced by each ray at each point. Points will be along the ray path and spaced by GRINStep.

The OPD retardance in nm/mm can be calculated by dividing the phase retardance by 2pi and multiplying by the wavelength in nm. 

**Parameters**:

* **density**: Density parameter used to generate the hexapolar pattern.
* **coordinateOption**: 1 to return positions in global coordinates. Otherwise positions are returned in local coordinates.
* **field**: Which field is used to generated the rays to be traced.


**Returns**:

Each entry represents a point along the ray path of one of the traced rays and provides its position, refractive index, and point retardance in radians / mm.


**Parameters**:

* int **density**
* int **coordinateOption**
* int **field**

Get the phase retardance (in radians) value at a given position and for a given propagation direction and step length.

The OPD retardance in nm can be calculated by dividing the phase retardance by 2pi and multiplying by the wavelength in nm. 

**Parameters**:

* **X**: In local coordinates and lens units.
* **Y**: In local coordinates and lens units.
* **Z**: In local coordinates and lens units.
* **L**: X direction cosine.
* **M**: Y direction cosine.
* **N**: Z direction cosine.
* **Step**: Step length in Lens Units.


**Returns**:

The maximum phase retardance in radians induced by stress birefringence.
