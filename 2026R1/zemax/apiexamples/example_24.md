# Example 24: Non-Sequential Detector Data
Ex 24: A basic non-sequential system is built with several different object types, and changes are made to object parameters through the INCERow interface.  Face type, coating, and scatter profile properties are set on volumetric objects.  A ray trace is run and a native OpticStudio detector is opened for reference. Then, the detector pixel data is extracted and plotted outside of OpticStudio.
