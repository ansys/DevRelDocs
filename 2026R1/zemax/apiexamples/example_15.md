# Example 15: Sequential Optimization
Ex 15: The quick focus and optimization tools are demonstrated.  A sample file is opened, and the ‘remove all variables’ and ‘quick focus’ tools are used.  Some variables and solves are specified, and a merit function is generated with the merit function wizard.  The local, global, and hammer optimizers are each run and closed in sequence.
