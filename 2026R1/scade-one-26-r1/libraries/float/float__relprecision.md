# RelPrecision

Returns the relative precision for comparing two numbers with an absolute precision.

## Interface

|Name|Kind|Type|Comment|
|----|:--:|----|-------|
|a|I|float|First operand|
|b|I|float|Second operand|
|e|I|float|Absolute precision<br>Range: Positive|
|precision|O|float|Relative precision|
