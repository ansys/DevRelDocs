# Speos RPC Changelog 2025 R2 SP03

## Secure Transport Protocols support

SpeosRPC_Server has been improved to support different secure transport protocols (mutual TLS, Unix Domain Socket, Windows Named User Authentication). Besides SpeosRPC version 2025 R2 SP03, this improvement applies retroactively to the 2025 R1 SP01, 2025 R2 and 2025 R2 SP01 versions of SpeosRPC. For more information on how to run SpeosRPC_Server, see [How to launch SpeosRPC Server?](../index.md#how-to-launch-speosrpc-server)
