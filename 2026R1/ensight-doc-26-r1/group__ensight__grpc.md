# ensight_grpc

<a id="group__ensight__grpc"></a>

Python wrapper for the core ensightservice.

This package defines the [EnSightGRPC](classensight__grpc_1_1_en_sight_g_r_p_c.md#classensight__grpc_1_1_en_sight_g_r_p_c) class which provides a simpler interface to the EnSight gRPC interface, including events and image steams.
