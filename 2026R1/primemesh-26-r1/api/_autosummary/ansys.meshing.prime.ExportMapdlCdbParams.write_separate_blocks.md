# ansys.meshing.prime.ExportMapdlCdbParams.write_separate_blocks

<a id="ansys.meshing.prime.ExportMapdlCdbParams.write_separate_blocks"></a>

#### *property* ExportMapdlCdbParams.write_separate_blocks *: [bool](https://docs.python.org/3.11/library/functions.html#bool)*

Controls whether element blocks should be written separately. When true, writes elements in separate blocks based on the format specified in separate_blocks_format_type. When false, writes all elements into a single block.

**This is a beta parameter**. **The behavior and name may change in the future**.
