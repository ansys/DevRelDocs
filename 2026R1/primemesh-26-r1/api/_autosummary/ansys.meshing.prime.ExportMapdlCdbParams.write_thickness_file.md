# ansys.meshing.prime.ExportMapdlCdbParams.write_thickness_file

<a id="ansys.meshing.prime.ExportMapdlCdbParams.write_thickness_file"></a>

#### *property* ExportMapdlCdbParams.write_thickness_file *: [bool](https://docs.python.org/3.11/library/functions.html#bool)*

Option to write a thickness file for spotweld fatigue analysis. If true, writes a file named [exportedFilename].cdb.thick.txt containing thickness information.

**This is a beta parameter**. **The behavior and name may change in the future**.
