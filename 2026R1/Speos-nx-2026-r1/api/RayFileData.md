# RayFileData Class

**Namespace:** `SpeosNX`

**Inherits from:** `object`

Represents Ray File Data.

## Description

Ray File Data class provides methods to get data from a ray file.



To obtain an instance of this class, refer to Speos Pattern Builder.

## Properties

| Name | Type | Description |
|------|------|-------------|
| `thisown` | `object` |  |
| `FluxInLumen` | `str` | Returns the flux value in lumen. |
| `FluxInWatt` | `str` | Returns the flux value in watt. |
| `HasFluxInLumen` | `str` | Returns true if the ray file has flux in lumen, otherwise returns false. |
| `HasFluxInWatt` | `str` | Returns true if the ray file has flux in watt, otherwise returns false. |
| `HasPower` | `str` | Returns true if the ray file has power, otherwise returns false. |
| `HasSpectrum` | `str` | Returns true if the ray file has spectrum, otherwise returns false. |
