# getFileSystemInfo


public [getFileSystemInfo](get-file-system-info.md)(uri: [String](https://docs.oracle.com/javase/8/docs/api/java/lang/String.html)): [FileSystemInfo](./../-file-system-info/index.md)

Gets the FileSystemInfo for a given URI. The URI doesn't necessarily need to exist.

#### Return

the FileSystemInfo for the given uri

#### Parameters



| | |
|---|---|
| uri | the URI of the file to get system info for |

#### Throws

| | |
|---|---|
| [ModelCenterException](./../-model-center-exception/index.md) | if an error is encountered generating the FileSystemInfo |
