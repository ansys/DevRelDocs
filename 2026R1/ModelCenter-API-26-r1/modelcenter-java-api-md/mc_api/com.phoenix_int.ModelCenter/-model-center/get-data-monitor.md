# getDataMonitor


public [getDataMonitor](get-data-monitor.md)(component: [String](https://docs.oracle.com/javase/8/docs/api/java/lang/String.html), index: [Variant](./../-variant/index.md)): [DataMonitor](./../-data-monitor/index.md)

gets a specific data monitor associated with a particular component

#### Return

a PHXCOMDataMonitor COM object representing the data monitor control

#### Parameters



| | |
|---|---|
| component | the name of the component under which to search for data monitors |
| index | the integer index (0-based index) or string title of the data monitor being sought |
