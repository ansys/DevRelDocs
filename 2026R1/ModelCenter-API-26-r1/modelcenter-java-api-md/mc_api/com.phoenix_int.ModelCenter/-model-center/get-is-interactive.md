# getIsInteractive


public [getIsInteractive](get-is-interactive.md)(): [Boolean](https://kotlinlang.org/api/latest/jvm/stdlib/kotlin/-boolean/index.html)

Determine if the current ModelCenter is running interactively - that is, if it has a GUI and a main window. Some methods will not work if the application is running in non-interactive (server or COM) mode.
