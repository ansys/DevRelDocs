# guiCloseAllMultiplexers


abstract fun [guiCloseAllMultiplexers](gui-close-all-multiplexers.md)(): [Boolean](https://kotlinlang.org/api/latest/jvm/stdlib/kotlin/-boolean/index.html)

Attempts to close the DataExplorers of all vizualization multiplexers.

#### Return

true if the user allowed all multiplexers to close, false if the user canceled the operation.
