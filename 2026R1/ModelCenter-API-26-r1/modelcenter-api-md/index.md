# ModelCenter COM API

The ModelCenter COM API provides a comprehensive interface for automating, customizing, and extending ModelCenter through Component Object Model (COM) technology. This API enables users to programmatically control ModelCenter, integrate with external applications, and develop custom plugins or automation scripts using languages such as Python, VBScript, or C#. With access to core ModelCenter objects, components, assemblies, and data, the COM API empowers users to streamline workflows, perform advanced analyses, and build robust integrations with third-party tools. This documentation details the available classes, interfaces, methods, and properties, serving as a reference for developers seeking to leverage the full power of ModelCenter in their engineering processes.
