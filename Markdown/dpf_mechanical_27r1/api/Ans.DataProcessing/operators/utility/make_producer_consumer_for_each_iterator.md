# *class* make_producer_consumer_for_each_iterator(try_generate_iterable: object = None, iterable: object = None, operator_to_iterate: object = None, pin_index: object = None, valueA: object = None, valueB: object = None, valueC1: object = None, valueC2: object = None, producer_op11: object = None, producer_op12: object = None, output_pin_of_producer_op12: object = None, input_pin_of_consumer_op12: object = None, consumer_op12: object = None, config: OperatorConfig = None)

Generates an iterator that can be consumed by the for_each operator.The chain of Operators are split into a first part : the producers and a second part : the consumers.Asynchronous buffers are indeed to connect the producers and the consumers.

available inputs: `try_generate_iterable` (bool) (optional), `iterable` (Any) (optional), `operator_to_iterate` (Operator), `pin_index` (Int32), `valueA` (Any), `valueB` (Any), `valueC1` (Any), `valueC2` (Any), `producer_op11` (Operator), `producer_op12` (Operator), `output_pin_of_producer_op12` (Int32), `input_pin_of_consumer_op12` (Int32), `consumer_op12` (Operator)

available outputs: `iterator` (Any)

**DPF Framework Reference:** [make_producer_consumer_for_each_iterator operator specification](https://developer.ansys.com/docs/dpf-framework-2027-r1/operator-specifications/utility/make_producer_consumer_for_each_iterator.md)

