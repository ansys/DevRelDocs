# ansys.meshing.prime.MergeZoneletsParams.merge_small_zonelets_with_neighbors

#### *property* MergeZoneletsParams.merge_small_zonelets_with_neighbors *: [bool](https://docs.python.org/3.11/library/functions.html#bool)*
