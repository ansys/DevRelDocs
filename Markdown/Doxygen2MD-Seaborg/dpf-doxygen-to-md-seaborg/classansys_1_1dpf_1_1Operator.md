# Class ansys::dpf::Operator

![][C++]
![][public]

**Definition**: `dpf\_api.h` (line 2215)

Wrap an elementary operation.

The [Operator](classansys_1_1dpf_1_1Operator.md#classansys_1_1dpf_1_1Operator) is the main object used to create, transform and stream the data. It can be seen as an integrated circuit in electronics with a range of pins in input and in output. The inputs and outputs attributes are then used to connect the operator with various sources, including other operators, data sources, fields, or provided data.
