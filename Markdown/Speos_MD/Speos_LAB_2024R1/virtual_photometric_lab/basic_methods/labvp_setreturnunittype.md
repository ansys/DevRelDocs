# SetReturnUnitType 

## Description 

Allows you to modify the unit of a map and to convert data from photometric to radiometric during the export of a spectral map in an extended map.

## Syntax 

*object*.SetReturnUnitType\(short iReturnUnitType\); As Short

- *object*: Virtual Photometric Lab object

- *iReturnUnitType*: returned unit type between OptisUnitTypeRadiometric = 0 and OptisUnitTypePhotometric = 1
