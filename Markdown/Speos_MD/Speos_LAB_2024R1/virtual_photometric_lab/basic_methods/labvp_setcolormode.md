# SetColorMode 

## **D**escription 

Sets the color mode of the map.

## Syntax 

*object*.SetColorMode\(short nColorMode\) As Short

- *object*: Virtual Photometric Lab object

- *nColorMode*: color mode to set

    0: OptisFalseColorModeBlackToWhiteWithColor

    1: OptisFalseColorModeWhiteToBlackWithColor

    2: OptisFalseColorModeBlackToWhiteWithGray

    3: OptisFalseColorModeWhiteToBlackWithGray

    4: OptisFalseColorModeBlueToRed

    5: OptisFalseColorModeRedToBlue

    6: OptisTrueColorMode

    7: OptisFalseColorModeRedLevels

    8: OptisFalseColorModeGreenLevels

    9: OptisFalseColorModeBlueLevels

    10:OptisFalseColorModeIso
