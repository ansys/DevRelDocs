# VisionModeEvaluationType 

## Description 

Allows you to select the way to evaluate the vision mode \(photopic, mesopic, scotopic\).

## Syntax 

*object*.VisionModeEvaluationType\(short nType\) As Short

- *object*: Virtual Human Vision Lab object

- *nType*: vision mode type

    0: mean value of luminance map

    1: max value of luminance map

    2: vision mode computed for each pixel
