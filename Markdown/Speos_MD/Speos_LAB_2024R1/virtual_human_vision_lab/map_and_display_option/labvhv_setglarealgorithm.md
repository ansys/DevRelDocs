# SetGlareAlgorithm 

## Description 

Allows you to select the glare algorithm.

## Syntax 

*object*.SetGlareAlgorithm\(short nType\) As Short

-   *object*: Virtual Human Vision Lab object

-   *nType*: glare algorithm

    0 : Vos

    1: Holladay


In order for the display to take these changes into account, you need to call the
[UpdateDisplayAndData](labvhv_updatedisplayanddata.md) method.
