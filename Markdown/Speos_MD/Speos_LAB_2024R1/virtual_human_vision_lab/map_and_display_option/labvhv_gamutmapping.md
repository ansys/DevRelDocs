# GamutMapping 

## Description 

Sets the gamut mapping and returns a value different from 0 if no error has occurred.

## Syntax 

*object*.GamutMapping\(short nGamutMapping\) As Short

-   *object*: Virtual Human Vision Lab object

-   *nGamutMapping*: 0 for **Gamut Clipping**, 2 for **Maintain Lightness and Hue**, 3 for **Maintain Hue**, 4 for **Colorimetric error**.
