# Show 

## Description 

Displays or hides the window of the Viewer/Editor and returns a value different than 0.

## Syntax 

*object*.Show\(short bShow\) As Short

-   *object*: BSDF -BRDF Anisotropic Viewer, Eulumdat Viewer, IES Viewer, User Material Editor, Virtual Photometric Lab or Virtual Human Vision Lab object

-   *bShow*: 0 for hide, else 1
