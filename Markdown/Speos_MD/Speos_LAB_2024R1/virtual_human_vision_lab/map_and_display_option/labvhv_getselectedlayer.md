# GetSelectedLayer 

## Description 

Gets a layer of a multi layer map. Returns 1 if no error has occurred \(map with layers\), 0 otherwise.

## Syntax 

*object*.GetSelectedLayer\(VARIANT\* ovLayerIndex\) As Short

- *object*: Virtual Photometric Lab or Virtual Human Vision Lab object

- *ovLayerIndex*: VARIANT of the index of the active layer \(-1 for all, -2 if many\)
