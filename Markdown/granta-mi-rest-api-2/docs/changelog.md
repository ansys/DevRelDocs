# Release Notes for Granta Cloud Connected Materials Data

New features, enhancements, bug fixes, and other changes.

## What's New
- Granta Material Picker now displays numeric data with an accuracy of 6 significant figures. <!--GWP-9658-->
- Granta Material Picker can now export up to 100 Material Model records in a single session, subject to the `maxRecordBatch` limit specified in the session request. <!--GWP-9759-->

## Enhancements and bug fixes

- Fixed an issue affecting display of your preferred theme in Granta Material Picker. <!--GWP-9893-->

## Upgrading from Version 1.0

This section summarizes all the breaking changes in the Granta Cloud Connected Materials Data platform and the Granta Integration Service API since the initial Version 1.0 release.
