## Discovery 

Ansys Discovery has journaling using Python scripting. It also has an Extensibility guide that uses .NET that can be used to create custom ribbons and tools. It is also possible to build custom Add-ins using C#.

* [Developer Portal Discovery Documentation](https://developer.ansys.com/docs/discovery)
  * Includes links to variety of assistant resources as well
* [Customaization in Ansys Discovery (White Paper)](https://www.ansys.com/en-gb/resource-center/white-paper/customization-ansys-discovery)
