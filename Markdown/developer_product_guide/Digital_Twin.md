# Digital Twin

## Twin Builder

Twin Builder, formerly known as Simplorer, supports journaling through Python or Visual Basic scripting and includes the PyAnsys library, PyTwin.

* [PyTwin documentation (PyAnsys)](https://twin.docs.pyansys.com/)
