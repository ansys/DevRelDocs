# invariant:von mises eqv (fields container)

Version: 0.0.0


## Description

Computes the element-wise Von-Mises criteria on all the tensor fields of a fields container.


## Inputs


| Input | Name | Expected type(s) | Description |
|-------|-------|------------------|-------------|
| <strong>Pin 0</strong> <br><span style="background-color:#d93025; color:white; padding:2px 6px; border-radius:3px; font-size:0.75em;">Required</span> | fields_container |[`fields_container`](../../core-concepts/dpf-types.md#fields-container) |  |
| <strong>Pin 13</strong> <br><span style="background-color:#d93025; color:white; padding:2px 6px; border-radius:3px; font-size:0.75em;">Required</span> | poisson_ratio |[`double`](../../core-concepts/dpf-types.md#double), [`int32`](../../core-concepts/dpf-types.md#standard-types) | Poisson ratio to be used in equivalent strain calculation. |
