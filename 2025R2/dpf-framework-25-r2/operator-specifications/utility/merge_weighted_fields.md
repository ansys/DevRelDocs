# utility:merge weighted fields

**Version: 0.0.0**

## Description

Assembles a set of fields into a unique one, applying a weight on the sum of the fields.

## Inputs

| Input | Name | Expected type(s) | Description |
|-------|-------|------------------|-------------|
| <strong>Pin -2</strong>|  sum_merge |[`bool`](../../core-concepts/dpf-types.md#standard-types) | Default is false. If true, redundant quantities are summed instead of being ignored. |
| <strong>Pin -1</strong>|  merged_support |[`abstract_field_support`](../../core-concepts/dpf-types.md#field-support) | Already merged field support. |
| <strong>Pin 0</strong> <br><span style="background-color:#d93025; color:white; padding:2px 6px; border-radius:3px; font-size:0.75em;">Required</span>|  fields |[`vector<shared_ptr<field>>`](../../core-concepts/dpf-types.md#vector<shared-ptr<field>>), [`field`](../../core-concepts/dpf-types.md#field), [`fields_container`](../../core-concepts/dpf-types.md#fields-container) | Either a fields container, a vector of fields to merge, or fields from pin 0 to ... |
| <strong>Pin 1000</strong> <br><span style="background-color:#d93025; color:white; padding:2px 6px; border-radius:3px; font-size:0.75em;">Required</span>|  weights |[`vector<shared_ptr<property_field>>`](../../core-concepts/dpf-types.md#vector<shared-ptr<property-field>>), [`property_field`](../../core-concepts/dpf-types.md#property-field) | Weights to apply to each field from pin 1000 to ... |

## Outputs

| Output |  Name | Expected type(s) | Description |
|-------|------|------------------|-------------|
|  **Pin 0**| merged_field |[`field`](../../core-concepts/dpf-types.md#field) |  |
