# utility:merge materials

Version: 0.0.0


## Description

Assembles a set of materials into a unique one.

| <strong>Pin 0</strong> <br><span style="background-color:#d93025; color:white; padding:2px 6px; border-radius:3px; font-size:0.75em;">Required</span> | materials |[`vector<shared_ptr<materials>>`](../../core-concepts/dpf-types.md#vector<shared-ptr<materials>>), [`materials`](../../core-concepts/dpf-types.md#materials) | A vector of materials to merge or materials from pin 0 to ... |
