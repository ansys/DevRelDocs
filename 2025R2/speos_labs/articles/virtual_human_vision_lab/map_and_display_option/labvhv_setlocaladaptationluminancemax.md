# SetLocalAdaptationLuminanceMax 

## Description 

Sets the maximum luminance value used for local adaptation. If this method is not used, the maximum luminance value used is the maximum luminance value on the XMP.

## Syntax 

*object*.SetLocalAdaptationLuminanceMax\(double dLuminanceMax\) As Short

- *object*: Virtual Human Vision Lab object

- *dLuminanceMax*: maximum luminance value
