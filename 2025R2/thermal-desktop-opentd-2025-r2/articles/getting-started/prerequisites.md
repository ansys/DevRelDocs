# Prerequisites

Before getting started with OpenTD, ensure that you meet the following prerequisites and technical requirements.

## Technical requirements

- To create OpenTD client programs that interact with Thermal Desktop, you'll need to use AutoCAD 2018 to 2024 inclusive. At present, OpenTD client programs will not work with AutoCAD 2025 and above.
- To create an OpenTD client or add-in program with AutoCAD 2018 to 2024, you must target .NET Framework 4.8.
- To create an OpenTD add-in program with AutoCAD 2025 and above, you must target .NET 8.
