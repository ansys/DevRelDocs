# invariant:scalar invariants (fields container)

## Version

0.0.0

## Changelog

- Version 0.0.0: Initial release.

## Description

Computes the element-wise invariants of all the tensor fields of a fields container.


## Inputs

| Input | Name | Expected type(s) | Description |
|-------|-------|------------------|-------------|
| <strong>Pin 0</strong> <br><span style="background-color:#d93025; color:white; padding:2px 6px; border-radius:3px; font-size:0.75em;">Required</span>|  fields_container |[`fields_container`](./../../core-concepts/dpf-types#fields-container) |  |


## Outputs

| Output |  Name | Expected type(s) | Description |
|-------|------|------------------|-------------|
|  **Pin 0**| fields_int |[`fields_container`](./../../core-concepts/dpf-types#fields-container) | stress intensity field |
|  **Pin 1**| fields_eqv |[`fields_container`](./../../core-concepts/dpf-types#fields-container) | stress equivalent intensity |
|  **Pin 2**| fields_max_shear |[`fields_container`](./../../core-concepts/dpf-types#fields-container) | max shear stress field |
