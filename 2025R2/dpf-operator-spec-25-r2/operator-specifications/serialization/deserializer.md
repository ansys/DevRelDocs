# serialization:deserializer

## Version

0.0.0

## Changelog

- Version 0.0.0: Initial release.

## Description

Takes a file generated by the serializer and deserializes it into DPF's entities.


## Inputs

| Input | Name | Expected type(s) | Description |
|-------|-------|------------------|-------------|
| <strong>Pin 0</strong> <br><span style="background-color:#d93025; color:white; padding:2px 6px; border-radius:3px; font-size:0.75em;">Required</span>|  file_path |[`string`](./../../getting-started/using-data-containers.md#string) | file path |


## Outputs

| Output |  Name | Expected type(s) | Description |
|-------|------|------------------|-------------|
|  **Pin 1**| any_output |[`any`](./../../getting-started/using-data-containers.md#any) | number and types of outputs corresponding of the inputs used in the serialization |

| Name| Expected type(s) | Default value | Description |
