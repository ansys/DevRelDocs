# result:rom data provider

## Version

0.0.0

## Changelog

- Version 0.0.0: Initial release.

## Description

Set the required data for the invariant terms computation (reduced matrices, lumped mass matrix, modes ...)

| Input | Name | Expected type(s) | Description |
