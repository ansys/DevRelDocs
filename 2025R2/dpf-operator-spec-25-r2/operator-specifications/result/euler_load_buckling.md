# result:euler load buckling

## Version

0.0.0

## Changelog

- Version 0.0.0: Initial release.

## Description

Computing Euler's Critical Load. Formula: Ncr = n*E*I*pi*pi /(L*L)

| Input | Name | Expected type(s) | Description |
|-------|-------|------------------|-------------|
