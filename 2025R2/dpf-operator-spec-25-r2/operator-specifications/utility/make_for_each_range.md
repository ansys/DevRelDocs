# utility:make for each range

## Version

0.0.0

## Changelog

- Version 0.0.0: Initial release.

## Description

Generate a range that can be consumed by the for_each operator


## Inputs

| Input | Name | Expected type(s) | Description |
|-------|-------|------------------|-------------|
| <strong>Pin -1</strong>|  try_generate_iterable |[`bool`](./../../core-concepts/dpf-types#standard-types) | if true, already iterable values connected in pin 3 like vectors, Scoping, TimefreqSupport, Containers and DataSources are split to iterate on it (default is true) |
| <strong>Pin 0</strong>|  iterable | | Iterable object, generated by make_for_each_range oeprator, that can be combined with the one currently generated. |
| <strong>Pin 1</strong> <br><span style="background-color:#d93025; color:white; padding:2px 6px; border-radius:3px; font-size:0.75em;">Required</span>|  operator_to_iterate |[`operator`](./../../core-concepts/dpf-types#operator) | Operator that must be reconnected with the range values. |
