# utility:data sources get attribute

## Version

0.0.0

## Changelog

- Version 0.0.0: Initial release.

## Description

A DataSources in pin 0 and a property name (string) in pin 1 are expected in input. An index refering to the property can also be provided.


## Inputs


| Input | Name | Expected type(s) | Description |
|-------|-------|------------------|-------------|
| <strong>Pin 0</strong> <br><span style="background-color:#d93025; color:white; padding:2px 6px; border-radius:3px; font-size:0.75em;">Required</span> | data_sources |[`data_sources`](../../core-concepts/dpf-types.md#data-sources) |  |
| <strong>Pin 1</strong> <br><span style="background-color:#d93025; color:white; padding:2px 6px; border-radius:3px; font-size:0.75em;">Required</span> | property_name |[`string`](../../core-concepts/dpf-types.md#string) | Accepted inputs are: 'file_path' (returns string), 'result_file_name' (returns string), 'domain_file_path' (returns string), 'domain_result_file_name' (returns string), 'num_keys' (returns int), num_result_key (returns int), num_file_path (returns int), 'num_result_file_path' (returns int), 'key_by_index' (returns string), 'result_key_by_index' (returns string), 'path_by_index' (returns string), 'path_key_by_index' (returns string). |
| <strong>Pin 2</strong> | property_index |[`int32`](../../core-concepts/dpf-types.md#standard-types) | Index for the property. Must be set for 'domain_file_path', 'domain_result_file_name' 'key_by_index', 'result_key_by_index', 'path_by_index' and 'path_key_by_index' properties. |
| <strong>Pin 3</strong> | property_key |[`string`](../../core-concepts/dpf-types.md#string) | Key to look for. Must be set for 'file_path' and 'domain_file_path' properties. |
| <strong>Pin 4</strong> | property_result_key |[`string`](../../core-concepts/dpf-types.md#string) | Result key to look for. Can be used for 'file_path', 'result_file_name', 'domain_file_path' and 'domain_result_file_name'. |

## Outputs


| Output | Name | Expected type(s) | Description |
|-------|------|------------------|-------------|
| **Pin 0** | property |[`string`](../../core-concepts/dpf-types.md#string), [`int32`](../../core-concepts/dpf-types.md#standard-types) | Property value. |
