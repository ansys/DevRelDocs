# NE {#ne .sone-topic}

Determines whether a value is different from another value, with respect to an absolute precision.

## Interface { .section}

|Name|Kind|Type|Comment|
|----|:--:|----|-------|
|a|I|float|First operand|
|b|I|float|Second operand|
|e|I|float|Absolute precision of the comparison

Range: Positive

|
|comp|O|bool|Result of the comparison|
