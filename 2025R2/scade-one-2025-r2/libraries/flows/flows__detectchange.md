# DetectChange {#detectchange .sone-topic}

Determines if the current value of the flow does not equal its value at the last cycle.

## Interface { .section}

|Name|Kind|Type|Comment|
|----|:--:|----|-------|
|in|I|polymorphic|Observed flow|
|out|O|bool|Detection flag|
