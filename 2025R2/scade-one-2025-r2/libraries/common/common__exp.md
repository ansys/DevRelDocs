# Exp {#common__exp .sone-topic}

Returns the exponential of the operand.

The implementation uses the C runtime library functions exp and expf.

## Interface { .section}

|Name|Kind|Type|Comment|
|----|:--:|----|-------|
|x|I|Numeric|Operand|
|v|O|Numeric|Result of the operation|
