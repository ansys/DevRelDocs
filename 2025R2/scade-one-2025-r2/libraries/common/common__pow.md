# Pow {#common__pow .sone-topic}

Returns the first operand raised to the power of the second one.

The implementation uses the C runtime library functions pow and powf.

## Interface { .section}

|Name|Kind|Type|Comment|
|----|:--:|----|-------|
|x|I|Numeric|First operand|
|y|I|Numeric|Second operand|
|v|O|Numeric|Result of the operation|
