# Sqrt {#common__sqrt .sone-topic}

Returns the square root of the operand.

The implementation uses the C runtime library functions sqrt and sqrtf.

## Interface { .section}

|Name|Kind|Type|Comment|
|----|:--:|----|-------|
|x|I|Float|Operand|
|v|O|Float|Result of the operation|
