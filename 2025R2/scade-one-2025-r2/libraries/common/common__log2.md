# Log2

Returns the common logarithm \(in base 2\) of the operand.

The implementation uses the C runtime library functions log2 and log2f.

## Interface

|Name|Kind|Type|Comment|
|----|:--:|----|-------|
|x|I|Float|Operand|
|v|O|Float|Result of the operation|
