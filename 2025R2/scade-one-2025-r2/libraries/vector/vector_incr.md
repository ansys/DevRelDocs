# Incr 

Addition of a scalar to all elements of a vector.

![](../images/vector__incr_01.svg)

## Interface 

|Name|Kind|Type|Comment|
|----|:--:|----|-------|
|N|P|N/A|Size of the vector|
|a|I|numeric^N|First input vector|
|x|I|numeric|Scalar|
|c|O|numeric^N|Output vector|
