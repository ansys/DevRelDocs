# FallingEdge {#fallingedge .sone-topic}

Detects a falling edge \(true to false transition\).

The output is true during the transition clock cycle. The output is initialized to false.

## Interface { .section}

|Name|Kind|Type|Comment|
|----|:--:|----|-------|
|in|I|bool|Observed flow|
|out|O|bool|Detection flag|
