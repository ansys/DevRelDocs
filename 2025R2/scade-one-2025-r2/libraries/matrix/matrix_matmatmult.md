# MatMatMult 

Multiplication of two matrices.

![](../images/matrix__matmatmult_00.svg)

## Interface 

|Name|Kind|Type|Comment|
|----|:--:|----|-------|
|N|P|N/A|Number of columns in first matrix and rows in second matrix|
|M|P|N/A|Number of rows in first matrix|
|P|P|N/A|Number of columns in second matrix|
|A|I|numeric^N^M|First input matrix|
|B|I|numeric^P^N|Second input matrix|
|C|O|numeric^P^M|Output matrix|
