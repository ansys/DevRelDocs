# ToBool {#tobool .sone-topic}

Returns an array of Boolean values from the input.

## Interface { .section}

|Name|Kind|Type|Comment|
|----|:--:|----|-------|
|N|P|N/A|Size of the bit-field|
|bf|I|unsigned<<N\>\>|Input bit-field|
|v|O|bool ^ N|Array of Boolean values|
