# dataCollectorCreated


abstract fun [dataCollectorCreated](data-collector-created.md)(tradeStudy: [TradeStudy2](../-trade-study2/index.md), dataCollector: [DataCollector](../-data-collector/index.md))

Called when the trade study creates a new data collector for it to use. The data collector should not be shown before being passed.

#### Parameters



| | |
|---|---|
| tradeStudy | the trade study associated with the data collector |
| dataCollector | the new data collector |
