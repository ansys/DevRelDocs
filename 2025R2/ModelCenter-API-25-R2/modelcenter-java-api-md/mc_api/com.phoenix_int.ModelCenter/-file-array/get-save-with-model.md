# getSaveWithModel


public [getSaveWithModel](get-save-with-model.md)(): [Boolean](https://kotlinlang.org/api/latest/jvm/stdlib/kotlin/-boolean/index.html)

Gets a flag indicating that the value of the file array should be saved with the model. The flag only applies if the file array is an output.

#### Return

true if the saveWithModel flag is true
