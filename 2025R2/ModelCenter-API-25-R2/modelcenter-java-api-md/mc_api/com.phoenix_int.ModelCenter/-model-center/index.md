# ModelCenter


public class [ModelCenter](index.md) : [PHXCOMObject](../../com.phoenix_int.ModelCenter.util/-p-h-x-c-o-m-object/index.md), [IModelCenter](../-i-model-center/index.md)

This is the wrapper class that handles the native COM calls to ModelCenter. The class is used by Java plug-ins to ModelCenter or by stand-alone Java programs to interact with ModelCenter. 

 Example: 

```kotlin

try
{
   ModelCenter mc = new ModelCenter();
   mc.loadModel( "d:\\test.pxc" );

   for ( int x = 0; x < 10; x++ )
   {
      System.out.print( "x = " + x );
      mc.setValue( "Model.Script.x", "" + x );
      System.out.println( ", y = " + mc.getValue( "Model.Script.y" ) );
   }

   mc.close();
}
catch ( ModelCenterException e )
{
   e.printStackTrace();
}

```
