# exportImage


abstract fun [exportImage](export-image.md)(hints: [String](https://docs.oracle.com/javase/8/docs/api/java/lang/String.html), filename: [String](https://docs.oracle.com/javase/8/docs/api/java/lang/String.html))

Tells the plug-in to export an image to a file.

#### Parameters



| | |
|---|---|
| hints | Hints to the page as to how to export. For now, this is not used, although in the future it can be used to express size, format, and &quot;thumbnail&quot; characteristics. |
| filename | The name of the file to create |
