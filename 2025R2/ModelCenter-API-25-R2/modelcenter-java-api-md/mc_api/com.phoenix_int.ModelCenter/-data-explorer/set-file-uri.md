# setFileUri


public [setFileUri](set-file-uri.md)(fileUri: [String](https://docs.oracle.com/javase/8/docs/api/java/lang/String.html))

Set the File URI loaded in this DataExplorer. This does not actually load the file, and is merely a container for data.

#### Parameters



| | |
|---|---|
| fileUri | the URI of the file loaded. |

#### Throws

| | |
|---|---|
| [ModelCenterException](../-model-center-exception/index.md) | if setting the URI fails |
