Detailed Description

When running a simulation, an IRockyParticleTransferScalars gives access to all particle transfer scalars defined in the project. Particle transfer scalars are special particle scalars whose values are reset automatically to zero at the beginning of a new time iteration.

If a custom scalar variable was defined during the setup of the module,
this method can be used to make the scalar store the maximum of a set of values. A common use of this method is, for instance, to find the maximum among a set of values associated to the contacts of a particle.
Parameters
