# Chapter 6: Quick Reference Guide to the Transport Subroutine Library

Users creating their own Ansys Chemkin Applications may need to access the
_Transport_ Subroutine Library directly to calculate transport properties.
This chapter is arranged by topical area to provide a quick reference to each
of the _Transport_ subroutines. In addition to the subroutine call list
itself, the purpose of the subroutine is briefly described.
