# MatchingMode

Determines how parts in the parts list are matched against parts in the AVL or
Part Library.

**Table  1.204. MatchingMode**

| **Name** | **Number** | **Description**                            |
| -------- | ---------- | ------------------------------------------ |
| Both     | 0          | Matches both part number and manufacturer. |
| Part     | 1          | Matches part number only.                  |
