# Example: Update PCB Modeling Properties

## Proto Request Example:

    
    
    {
      "project": "Tutorial Project",
      "ccaNames": [
        "Main Board"
      ],
      "analyses": [
        {
          "type": 6,
          "modelType": 1,
          "modelingRegionEnabled": true,
          "pcbMaterialModel": 2,
          "pcbMaxMaterials": 5,
          "pcbElemOrder": 1,
          "pcbMaxEdgeLength": 0.5,
          "pcbMaxEdgeLengthUnits": "mm",
          "pcbMaxVertical": 1,
          "pcbMaxVerticalUnits": "mm",
          "quadsPreferred": true
        }
      ]
    }
