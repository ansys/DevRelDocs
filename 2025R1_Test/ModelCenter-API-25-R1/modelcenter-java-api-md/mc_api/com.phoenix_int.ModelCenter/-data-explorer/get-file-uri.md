# getFileUri


public [getFileUri](get-file-uri.md)(): [String](https://docs.oracle.com/javase/8/docs/api/java/lang/String.html)

Get the File URI loaded into this DataExplorer.

#### Return

the URI of the file loaded in this DataExplorer, empty string if no file is currently loaded.

#### Throws

| | |
|---|---|
| [ModelCenterException](../-model-center-exception/index.md) | if getting the URI fails |
