public [setRunInParallel](set-run-in-parallel.md)(runInParallel: [Boolean](https://kotlinlang.org/api/latest/jvm/stdlib/kotlin/-boolean/index.html))

Tells the job manager whether or not to run the jobs in parallel

| | |
|---|---|
