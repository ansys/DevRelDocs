# getType


public [getType](get-type.md)(): [String](https://docs.oracle.com/javase/8/docs/api/java/lang/String.html)

Gets the type of the variable. Valid types include: 

- double
- long
- string
- boolean
- reference
- file
- quadFacet
- sor
- nurbs
- bspline
- ruled
- skinned
- node
- double[]
- long[]
- string[]
- boolean[]
- reference[]

#### Return

the variable type
