# fromString


public [fromString](from-string.md)(str: [String](https://docs.oracle.com/javase/8/docs/api/java/lang/String.html))

Sets the value for the variable from a string. For 1D arrays, the specification is of the form '1,2,3'. For nD arrays, the specification is of the form 'bounds[2,2,2] {1,2,3,4,5,6,7,8}'. String arrays may optionally have the elements quoted in the form 'bounds[2,3] {&quot;a&quot;, &quot;b,c&quot;, &quot;d&quot;, &quot;&quot;, &quot;e&quot;, &quot;f&quot;}'
