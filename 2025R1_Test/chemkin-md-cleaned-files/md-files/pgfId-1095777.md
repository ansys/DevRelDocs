    SUBROUTINE CKXMIN (X, NN, XMIN, IMIN)
    Returns the minimum value in an array and its location in the array.

    INPUT
    X(*)      - Real array.
    NN        - Integer scalar; size of X.
    OUTPUT
    XMIN      - Real scalar.
    IMIN      - Integer scalar; location in X of XMIN.
