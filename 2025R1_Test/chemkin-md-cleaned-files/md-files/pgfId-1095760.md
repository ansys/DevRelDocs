    SUBROUTINE CKXMAX (X, NN, XMAX, IMAX)
    Returns the maximum value in an array and its location in the array.

    INPUT
    X(*)      - Real array.
    NN        - Integer scalar; size of X.
    OUTPUT
    XMAX      - Real scalar.
    IMAX      - Integer scalar; location in X of XMAX.
