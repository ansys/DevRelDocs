    SUBROUTINE CKSCAL (ARRAY, NN, SCAL)
    Utility to scale the real members of an array.

    INPUT
    ARRAY(*)  - Real array.
    NN        - Integer scalar; the size of ARRAY.
    SCAL      - Real scalar; the multiplier for ARRAY members.
