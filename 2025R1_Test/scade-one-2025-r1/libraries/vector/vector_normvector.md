# NormVector {#vector__normvector .sone-topic}

Normalization of a vector.

Returns a vector of length 1 with the same direction. If the vector has length zero, returns the same vector.

![](../images/vector__normvector_00.svg)

## Interface { .section}

|Name|Kind|Type|Comment|
|----|:--:|----|-------|
|N|P|N/A|Size of the vector|
|a|I|float^N|Input vector|
|b|O|float^N|Normalized vector|
