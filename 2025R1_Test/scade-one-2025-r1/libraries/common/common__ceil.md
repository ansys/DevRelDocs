# Ceil

Returns the smallest integer greater than or equal to the operand.

The implementation uses the C runtime library functions ceil and ceilf.

## Interface

|Name|Kind|Type|Comment|
|----|:--:|----|-------|
|x|I|Float|Operand|
|v|O|Float|Result of the operation|
