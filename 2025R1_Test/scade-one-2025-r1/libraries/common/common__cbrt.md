# Cbrt

Returns the cube root of the operand.

The implementation uses the C runtime library functions cbrt and cbrtf.

## Interface

|Name|Kind|Type|Comment|
|----|:--:|----|-------|
|x|I|Float|Operand|
|v|O|Float|Result of the operation|
