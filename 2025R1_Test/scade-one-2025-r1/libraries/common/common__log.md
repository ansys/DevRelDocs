# Log

Returns the natural logarithm \(in base e\) of the operand.

The implementation uses the C runtime library functions log and logf.

## Interface

|Name|Kind|Type|Comment|
|----|:--:|----|-------|
|x|I|Float|Operand|
|v|O|Float|Result of the operation|
