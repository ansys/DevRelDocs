# Class SpeosNX::FeatureSimulation

![][Python]
![][public]

**Definition**: `SpeosNX.py` (line 11780)

Represents a Speos simulation feature.

A base class for all Speos simulation features.<br/>






This is an abstract class that provides the basic functionality common to all simulation features.<br/>
 This is an abstract class, and cannot be instantiated.<br/>
 To obtain an instance of this class, refer to the simulation builders.<br/>
