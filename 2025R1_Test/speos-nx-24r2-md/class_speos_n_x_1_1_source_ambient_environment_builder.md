### Property WhitePointType

![][public]
![][static]

**Definition**: `SpeosNX.py` (line 9954)

```python
WhitePointType
```

Gets or sets the white point type of the standard illuminant.

Prerequisite: The ColorSpace property must be 2.<br/>






The values are:<br/>
 0 - D65. Uses a standard daylight illuminant that provides accurate color perception and evaluation.<br/>
 1 - D50. Uses a natural, horizon light.<br/>
 2 - C. Uses an average daylight illuminant.<br/>
 3 - E. Uses an illuminant that gives equal weight to all wavelengths.<br/>
 4 - User defined. Allows to edit the Color Coordinates of the white point.<br/>






**Value type**: Integer.<br/>






The default value is 0.
