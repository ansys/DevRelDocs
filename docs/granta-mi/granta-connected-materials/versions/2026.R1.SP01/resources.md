# Resources

Resources to achieve a consistent look and feel when integrating Granta Material Picker into your application.

## Names and Acronyms

The name of the hosted product is **Granta MI Connected Materials**.

Use the following names when referring to components of the system:

| Name  |  Acronym |
|---|---|
| Granta Material Picker   | GMP  |
| Granta Integration Service  | GIS  |
| Granta Material Model Service | GMMS  |
| Material Model Definitions | MMDs  |
| Material Model Outputs | MMOs  |

## User Interface Terminology

When adding controls to launch Granta Material Picker, use the following text for tooltips and menu items:

**Import from Granta MI Connected Materials**

## Icons

All icons are available in all Ansys Design Language themes, and provided in standard sizes as .bmp and .png files in [Granta_Connected_Materials_icons.zip](resources/Granta_Connected_Materials_icons.zip).
