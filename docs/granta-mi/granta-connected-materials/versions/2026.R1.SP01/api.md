# Granta Integration Service

The Integration Service provides a REST API over HTTPS for integrating materials data into host applications.

The API reference documentation is available here:

[Granta Integration Service](../granta-integration-service-1-0-0)

## Material Model Service

The Material Model Service provides a REST API over HTTPS for retrieving material model data from Granta MI Connected Materials.

[https://grantamaterials.ansys.com/mi_servicelayer/proxy/v1.svc/mms/swagger/](https://grantamaterials.ansys.com/mi_servicelayer/proxy/v1.svc/mms/swagger/index.html)

Note: If you cannot access the Swagger documentation, ensure that you have a valid Ansys ID, and that you have logged in to the [MI Service Layer homepage](https://grantamaterials.ansys.com/mi_servicelayer/).
