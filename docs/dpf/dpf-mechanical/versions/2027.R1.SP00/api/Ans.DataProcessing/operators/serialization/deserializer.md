# *class* deserializer(stream_type: object = None, file_path: object = None, config: OperatorConfig = None)

Takes a file generated by the serializer and deserializes it into DPF's entities.

available inputs: `stream_type` (Int32), `file_path` (string)

available outputs: `any_output1` (Any), `any_output2` (Any)

**DPF Framework Reference:** [deserializer operator specification](https://developer.ansys.com/docs/dpf-framework-2027-r1/operator-specifications/serialization/deserializer.md)

**Parameters:**

* **stream_type**
* **file_path**
* **config**

**Example:**

```python
op = deserializer()

op = deserializer(stream_type=my_stream_type,file_path=my_file_path)
```

## Inputs

### stream_type

0 for ASCII (default), and 1 for binary

**Type:** *LinkableInput*

### file_path

file path

**Type:** *LinkableInput*

## Outputs

### any_output1

number and types of outputs corresponding of the inputs used in the serialization

**Type:** *LinkableOutput*

### any_output2

number and types of outputs corresponding of the inputs used in the serialization

**Type:** *LinkableOutput*
