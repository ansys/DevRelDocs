# Class ansys::dpf::PropertyFieldsContainer

<a id="classansys-dpf-propertyfieldscontainer"></a>

![][C++]
![][public]



Contains a group of property fields. The [PropertyFieldsContainer](classansys-dpf-propertyfieldscontainer.md#classansys-dpf-propertyfieldscontainer) is a set of property fields ordered through labels and ids. It is the external-API equivalent of the internal CPropertyFieldsContainer. 

**See also**: [PropertyField](classansys-dpf-propertyfield.md#classansys-dpf-propertyfield)

**Inherits from**:

* ansys::dpf::CollectionBase
