# Example 01: New File and QuickFocus
Ex 1: New lens file is created and saved in samples directory.  Changes are made to aperture value, field point is added, and wavelength preset for FdC is selected.  Comments, thicknesses, and radii are changed in the LDE; an F# solve is placed on the last surface's radius. Finally, the quick focus tool is opened, executed, and closed.

    Input:	 	None

    Output:	 	API\e01_new_file_and_quickfocus.zos
