# Example 25: Source Spectrum Diffraction Grating
Ex 25: A non-sequential system with two black body sources of different temperatures and a diffraction grating is built. The system is setup with multi-configuration to control the number of analysis rays traced for each source.  Only a single source has analysis rays during a ray trace.  The true color results are plotted for each configuration.
