# Example 10: NSC ZRD Filter String
Ex 10: A sample file is opened, all detectors are cleared, and a ray trace is run while saving a ZRD file.  The ZRD file is played back on a detector viewer window, while a filter string is applied to the detector.  The number of ray hits and total flux is retrieved from the detector for each pixel.  For Premium only, a Ray Path Analysis window is opened, the ZRD is loaded and the output is saved to a text file.

    Input:		Non-Sequential\Ray Splitting\Beam splitter.zos

    Output:		API\e10_NSC_ZRD_filter_string.zos
                API\e10_API_Raytrace.ZRD
                API\e10_RayPathAnalysis.txt (Premium Only)
