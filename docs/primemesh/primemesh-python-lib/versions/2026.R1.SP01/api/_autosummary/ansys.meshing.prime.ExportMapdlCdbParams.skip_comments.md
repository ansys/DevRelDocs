# ansys.meshing.prime.ExportMapdlCdbParams.skip_comments

<a id="ansys.meshing.prime.ExportMapdlCdbParams.skip_comments"></a>

#### *property* ExportMapdlCdbParams.skip_comments *: [bool](https://docs.python.org/3.11/library/functions.html#bool)*

Option to skip export of comments to the exported file. When the value is true, skips writing comments. When the value is false, writes comments to the exported file. The default value is false.

**This is a beta parameter**. **The behavior and name may change in the future**.
