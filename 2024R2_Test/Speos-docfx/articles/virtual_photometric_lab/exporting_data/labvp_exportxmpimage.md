# ExportXMPImage 

## Description 

Saves the image display of the map in a file. Returns a value different from 0 if no error has occurred.

## Syntax 

*object*.ExportXMPImage\(BSTR bstrFileName, int iFormat\) As Short

- *object*: Virtual Photometric Lab or Virtual Human Vision Lab object

- *bstrFileName*: file name to save the image

- *iFormat*: image format \(0: BMP, 1: PNG, 2 :TIFF, 3: JPG\)
