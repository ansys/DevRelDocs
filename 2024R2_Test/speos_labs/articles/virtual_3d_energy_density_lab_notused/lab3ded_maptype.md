# MapType

## Description

Gets the type of map.

## Syntax

*object*.MapType\(\) As Long

- *object*: 3D Energy Density Lab object

- Returned value:

    0 : OptisMapTypeBasic

    1 : OptisMapTypeLaser

    2 : OptisMapTypeSpectral

    3 : OptisMapTypeExtended

    4 : OptisMapTypeUnknown

    5 : OptisMapTypeGainMatrix

    6 : OptisMapTypeRays

    7 : OptisMapTypeFTM
