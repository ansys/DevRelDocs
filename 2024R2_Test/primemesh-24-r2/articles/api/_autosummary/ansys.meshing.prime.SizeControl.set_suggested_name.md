#### SizeControl.set_suggested_name(name)

  : Suggested name for the size control.
* **Returns:**
  [`SetNameResults`](ansys.meshing.prime.SetNameResults.md#ansys.meshing.prime.SetNameResults)

>>> size_control.set_suggested_name("control1")
