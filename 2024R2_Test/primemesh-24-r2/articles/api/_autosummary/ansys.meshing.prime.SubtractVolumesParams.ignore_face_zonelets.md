# ansys.meshing.prime.SubtractVolumesParams.ignore_face_zonelets



#### *property* SubtractVolumesParams.ignore_face_zonelets *: [Iterable](https://docs.python.org/3.11/library/typing.html#typing.Iterable)[[int](https://docs.python.org/3.11/library/functions.html#int)]*

Face zonelet ids that subtract volumes should not remove (for example, periodic or fluid cap zonelets). If ignore face zonelets are provided, then the target volumes after subtract operation need to be recomputed.
