# ansys.meshing.prime.LabelExportParams.label_expression_for_nodal_components

#### *property* LabelExportParams.label_expression_for_nodal_components *: [str](https://docs.python.org/3.11/library/stdtypes.html#str)*

Label expression to export matching labels as Nodal Components in CDB. Non-matching labels will be exported as Element Components.

**This is a beta parameter**. **The behavior and name may change in the future**.
