# bit_count

<a id="ansys.meshing.prime.WrapRegion.bit_count"></a>

#### WrapRegion.bit_count(/)

Number of ones in the binary representation of the absolute value of self.

Also known as the population count.

```pycon
>>> bin(13)
'0b1101'
>>> (13).bit_count()
3
```
