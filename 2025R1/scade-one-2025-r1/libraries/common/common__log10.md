# Log10

Returns the common logarithm \(in base 10\) of the operand.

The implementation uses the C runtime library functions log10 and log10f.

## Interface

|Name|Kind|Type|Comment|
|----|:--:|----|-------|
|x|I|Float|Operand|
|v|O|Float|Result of the operation|
