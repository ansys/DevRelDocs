# ConcatRows 

Concatenation of two matrices' rows.

![](../images/matrix__concatrows_00.svg)

## Interface 

|Name|Kind|Type|Comment|
|----|:--:|----|-------|
|N|P|N/A|Number of columns in matrix|
|M1|P|N/A|Number of rows in first matrix|
|M2|P|N/A|Number of rows in second matrix|
|A1|I|numeric^N^M1|First input matrix|
|A2|I|numeric^N^M2|Second input matrix|
|B|O|numeric^N^\(M1+M2\)|Output matrix|
