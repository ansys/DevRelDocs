# Scale 

Multiplication of all elements of a vector by a matrix.

![](../images/matrix__scale_00.svg)

## Interface 

|Name|Kind|Type|Comment|
|----|:--:|----|-------|
|N|P|N/A|Number of columns in matrix|
|M|P|N/A|Number of rows in matrix|
|A|I|numeric^N^M|Input matrix|
|x|I|numeric|Scalar|
|B|O|numeric^N^M|Output matrix|
