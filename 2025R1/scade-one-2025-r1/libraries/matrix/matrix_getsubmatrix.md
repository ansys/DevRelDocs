# GetSubMatrix {#matrix__getsubmatrix .sone-topic}

Returns the sub matrix of size Ms by Ns, starting at row Midx and column Nidx.

## Interface { .section}

|Name|Kind|Type|Comment|
|----|:--:|----|-------|
|N|P|N/A|Number of columns in input matrix|
|M|P|N/A|Number of rows in input matrix|
|Nidx|P|N/A|Starting column of sub matrix|
|Ns|P|N/A|Number of columns in sub matrix|
|Midx|P|N/A|Starting row of sub matrix|
|Ms|P|N/A|Number of rows in sub matrix|
|A|I|numeric^N^M|Input matrix|
|S|O|numeric^Ns^Ms|Output sub matrix|
