# VecMatMult 

Multiplication of a vector and a matrix.

![](../images/matrix__vecmatmult_00.svg)

## Interface 

|Name|Kind|Type|Comment|
|----|:--:|----|-------|
|N|P|N/A|Number of columns in matrix and size of output vector|
|M|P|N/A|Number of rows in matrix and size of input vector|
|v|I|numeric^M|Input vector|
|A|I|numeric^N^M|Input matrix|
|w|O|numeric^N|Output vector|
