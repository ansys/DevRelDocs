# ConcatCols 

Concatenation of two matrices' columns.

![](../images/matrix__concatcols_00.svg)

## Interface 

|Name|Kind|Type|Comment|
|----|:--:|----|-------|
|N1|P|N/A|Number of columns in first matrix|
|N2|P|N/A|Number of columns in second matrix|
|M|P|N/A|Number of rows in matrix|
|A1|I|numeric^N1^M|First input matrix|
|A2|I|numeric^N2^M|Second input matrix|
|B|O|numeric^\(N1+N2\)^M|Output matrix|

**Parent topic:**[Math::Matrix](./math_matrix.md)
