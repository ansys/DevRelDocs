# Introduction

optiSLang provides various Python modules that can be used to extend functionality. Python examples can be found [here]( https://ansyshelp.ansys.com/public/account/secured?returnurl=/Views/Secured/corp/v251/en/opti_tut/opti_tut_examples.html ).

This documentation provides an overview of the optiSLang Python API. This API can be used to create and modify optiSLang projects using Python scripting as well as to customize certain optiSLang functionality.
