# Floor

Returns the largest integer less than or equal to the operand.

The implementation uses the C runtime library functions floor and floorf.

## Interface

|Name|Kind|Type|Comment|
|----|:--:|----|-------|
|x|I|Numeric|Operand|
|v|O|Numeric|Result of the operation|
