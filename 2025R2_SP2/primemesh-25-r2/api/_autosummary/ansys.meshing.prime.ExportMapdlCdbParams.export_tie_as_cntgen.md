# ansys.meshing.prime.ExportMapdlCdbParams.export_tie_as_cntgen

<a id="ansys.meshing.prime.ExportMapdlCdbParams.export_tie_as_cntgen"></a>

#### *property* ExportMapdlCdbParams.export_tie_as_cntgen *: [bool](https://docs.python.org/3.11/library/functions.html#bool)*

Option to export ties as cntgen. When true, translates ties and contact pairs into compact cntgen blocks in the exported file. The default value is false.

**This is a beta parameter**. **The behavior and name may change in the future**.
