# ansys.meshing.prime.ExportMapdlCdbParams.export_coupling_as_sfcgen

<a id="ansys.meshing.prime.ExportMapdlCdbParams.export_coupling_as_sfcgen"></a>

#### *property* ExportMapdlCdbParams.export_coupling_as_sfcgen *: [bool](https://docs.python.org/3.11/library/functions.html#bool)*

Option to export coupling as sfcgen. When true, translates kinematic or distributing coupling into compact sfcgen blocks in the exported file. The default value is false.

**This is a beta parameter**. **The behavior and name may change in the future**.
